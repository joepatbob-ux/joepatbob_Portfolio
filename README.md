# joepatbob.com

Portfolio site for Joseph Patrick Roberts, Principal Product Designer.

## Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Deployed on Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Before running, add fonts

Place these files in `/public/fonts/`:
- `AlteHaasGroteskBold.woff2`
- `AlteHaasGroteskRegular.woff2`
- `JetBrainsMono-Bold.woff2`
- `JetBrainsMono-Regular.woff2`

## Adding images

Place images in `/public/images/` — see `DESIGN_SPEC.md` for the full list.

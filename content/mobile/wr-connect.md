WR Connect configures ignition control boards for contractors working under time pressure on equipment that can't be powered up until the configuration is right. The old way was DIP switches, cryptic codes, and documentation written for engineers, where one wrong setting can fry a component before the system ever runs. Because the board sits unpowered during install, live-board options like Bluetooth were off the table, which pointed straight to NFC. Read pulls the current configuration, you edit a plain list, Write pushes it back. No power, no DIP switches, no codes. All the complexity that used to live on the board moved into the app, where a contractor can get it right before touching hardware.

The app carries saved profiles for repeat jobs, so it doesn't force everyone through the first-timer path. Once the board powers on, the same tool switches to a Bluetooth diagnostic channel for live fault codes and system status. One protocol for setup, another for diagnostics, each fit to the moment. WR Connect won Gold for Most Valuable App at the 2023 ACHR News Dealer Design Awards. The better signal came later: the core functionality got folded into Copeland Mobile and the standalone app was retired.

**Not every contractor needs the beginner path**

Saved profiles come two ways, some shipped by Copeland, some built by the contractor. Someone installing the same board all week skips Read entirely and writes a known config straight to it. The app carries all three paths without pushing everyone through the one built for a first-timer.

**One tool, every board a tech might see**

Copeland makes replacement boards for most competitor models a contractor runs into, and WR Connect drives all of them. The pattern scaled past the original board too: a later universal ignition module, configured through the same app, replaces more than 325 individual part numbers with one board and an NFC pairing flow, cutting what a contractor needs to stock on the truck. Same tool no matter which board is going in, instead of a different app or process per manufacturer.

WR Connect configures ignition control boards for contractors working under time pressure on equipment that can't be powered up until the configuration is right — traditionally DIP switches, cryptic codes, and documentation written for engineers, where one wrong setting can fry a component before the system ever runs. Because the board sits unpowered during install, live-board options like Bluetooth were off the table, which pointed to NFC: Read pulls the current configuration, you edit a plain list, Write pushes it back — no power, no DIP switches, no codes. All the complexity that used to live on the board moved into the app, where a contractor can get it right before touching hardware.

The app also carries saved profiles for repeat jobs, so it doesn't force every contractor through the first-timer path, and once the board powers on, the same tool switches to a Bluetooth diagnostic channel for live fault codes and system status — one protocol for setup, another for diagnostics, each fit to the moment. WR Connect won Gold for Most Valuable App at the 2023 ACHR News Dealer Design Awards; the better signal was that its core functionality later got folded into Copeland Mobile, and the standalone app was retired.

**Not every contractor needs the beginner path**

For repeat jobs, the app keeps saved profiles — some shipped by Copeland, some built by the contractor. Someone who installs the same board all week can skip the Read step entirely and write a known configuration straight to it; a technician who already knows the change can do the same. The app carries all three paths without pushing everyone through the one built for a first-timer.

**One tool, every board a tech might see**

Copeland makes replacement boards for most of the competitor models a contractor runs into in the field, and WR Connect drives all of them — so it's the same tool no matter which board is being swapped in, instead of a different app or process per manufacturer.
